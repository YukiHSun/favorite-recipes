鶏のからあげ

メイン調理 25分2人分

材料

  • 鶏もも肉 … 300g
  • 醤油 … 大さじ2
  • 酒 … 大さじ1
  • おろしにんにく・生姜 … 各小さじ1
  • 片栗粉 … 大さじ4
  • 揚げ油 … 適量

作り方

  1. 鶏もも肉を一口大に切る。
  2. ポリ袋に醤油・酒・にんにく・生姜と鶏肉を入れ、10分漬け込む。
  3. 片栗粉をまぶし、170℃の油で4分揚げる。
  4. 一度取り出し、190℃で1分二度揚げしてカリッと仕上げる。

メモ: 二度揚げがサクサクの決め手。レモンを添えると◎。

豚の生姜焼き

メイン調理 15分2人分

材料

  • 豚ロース薄切り … 250g
  • 玉ねぎ … 1/2個
  • おろし生姜 … 大さじ1
  • 醤油・みりん・酒 … 各大さじ2
  • 砂糖 … 小さじ1
  • サラダ油 … 大さじ1

作り方

  1. 玉ねぎは薄切り、調味料をすべて混ぜておく。
  2. フライパンに油を熱し、豚肉と玉ねぎを炒める。
  3. 合わせ調味料を加え、汁気が少し残るまで煮絡める。
  4. 皿に盛り、千切りキャベツを添えて完成。

メモ: 豚肉に先に片栗粉を薄くまぶすと、タレの絡みが良くなる。

ほうれん草のお浸し

副菜調理 10分2人分

材料

  • ほうれん草 … 1束
  • だし汁 … 100ml
  • 醤油 … 大さじ1
  • みりん … 小さじ1
  • かつお節・白ごま … 各適量

作り方

  1. ほうれん草を沸騰した湯で30秒茹で、冷水にとって水気を絞る。
  2. 4cm幅に切り、だし汁・醤油・みりんの合わせ地に15分浸す。
  3. 器に盛り、かつお節と白ごまを散らす。

メモ: 茹で時間は短めがコツ。鮮やかな緑が残る。

基本のミートソースパスタ

メイン調理 30分2人分

材料

  • 合いびき肉 … 200g
  • 玉ねぎ … 1個（みじん切り）
  • にんにく … 1片（みじん切り）
  • カットトマト缶 … 1缶（400g）
  • コンソメ … 小さじ2／砂糖 … 小さじ1
  • 塩・黒こしょう … 適量
  • スパゲッティ … 200g

作り方

  1. 鍋にオリーブ油とにんにくを熱し、玉ねぎを飴色まで炒める。
  2. ひき肉を加え、強火でしっかり焼き色を付ける。
  3. トマト缶・コンソメ・砂糖を入れ、弱火で15分煮込む。
  4. 塩・黒こしょうで味を整える。
  5. 茹でたスパゲッティとソースを和え、皿に盛る。

メモ: ひき肉は触りすぎず、塊のまま焼くと香ばしさが出る。

バスクチーズケーキ

スイーツ調理 50分15cm型

材料

  • クリームチーズ … 200g
  • グラニュー糖 … 70g
  • 卵 … 2個
  • 生クリーム … 200ml
  • 薄力粉 … 大さじ1

作り方

  1. オーブンを240℃に予熱する。
  2. 室温に戻したクリームチーズを滑らかに混ぜ、砂糖を加える。
  3. 溶き卵→生クリーム→薄力粉の順に加え、その都度よく混ぜる。
  4. クッキングシートを敷いた型に流し、240℃で25分焼く。
  5. 表面がしっかり焦げたら取り出し、粗熱→冷蔵庫で半日冷やす。

メモ: 焼きすぎず、真ん中がわずかに揺れる状態で止めると濃厚に。
